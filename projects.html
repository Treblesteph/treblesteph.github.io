  • device_hubnetwork evolution

    Biological gene expression can be modelled by abstract networks where positive/negative interactions represent gene activation/repression respectively. By observing the dynamic behaviour of these networks over time we can see biologically functional behaviours such as circadian rhythms and bet-hedging.

    This study seeks to discover small robust networks that perform a specified function successfully.

    We start by creating an initial population of random networks, we then rank them according to some fitness function relating to the biological function we are interested in, for example if it were a circadian rhythm then we could rank them according to how well they reliably switched on a specific gene at the same time each day. Then reproduction is performed by way of emulating biological evolution with the network as the unit of reproduction. The fittest networks are likely to reproduce more, and the least fit networks are killed off. Reproduction can occur through recombination or clonally, and mutations occur randomly across the whole population from one generation to the next. Then the whole process is repeated until the fittest network is either optimally fit according to the fitness function, or if it no longer increases in fitness from one generation to the next.

    I am doing this project in the open, so take a look at the github repo where there is a more in-depth description of the project in the README.

  • schoolscrapermaker

    The Content Mine project led by Peter Murray-Rust aims to set the academic literature free so that it is accessible to everyone. Their approach involves crawling the sources of literature on the web, scraping articles and their meta data, and extracting their facts or other entities. The scraping step is done with the quickscrape tool, and scrapers for individual publishers are built and maintained by the community.

    Currently a contributor makes a scraper for their journal of choice by creating a scraperJSON. However this process is difficult for people who are not used to working with code.

    The scrapermaker is a Chrome extension which guides you through a series of simple steps to make your own scraper easily.

    I am doing this project in the open, so take a look at the github repo where there is a more in-depth description of the project in the README.
