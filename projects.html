  • device_hubnetwork evolution

    An example evolved network

    This network is a biological clock. Each of the four genes, A, B, C, and D, can interact with each other by either activating (arrows) or repressing (T-lines). Each interaction has an associated lag (to account for transcription, translation, and protein-folding processes), these are marked next to the interaction lines. The effect of multiple incoming interactions on a gene is determined by a logic gate, this is simply an 'and' or 'or' gate currently. Finally, each gene can be activated by light (indicated by a sunshine), with an associated lag. The fitness of the network is shown (between 0 and 1, and the lower the better), and so are the number of light inputs, and the number of feedback loops. The number of feedback loops is calculated using a depth first search (DFS) algorithm. It was really fun to write, so I'll try to write a blog post about it. I am doing this project in the open, so take a look at the github repo, and feel free to star or fork :)
